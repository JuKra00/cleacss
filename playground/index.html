cleacss

This page is written in cleacss and is intended as a little guideline to explore how the framework is intended to be used. You can toggle the grid overlay in the bottom right corner to see the grid and flow layout in action.

The framework comes with sensible defaults for html elements and helper classes to make development fast and easy. At base level, cleacss provides a set of css variables with opinionated default values. There variables are the basis for all base styling, components and utilities.
