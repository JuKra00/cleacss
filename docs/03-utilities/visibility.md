You can use helper classes to hide elements on certain breakpoints.


| Class name          | Breakpoint |
| ------------------- | ---------- |
| `is-hidden`         | `all`      |
| `is-hidden:s`       | `> s`      |
| `is-hidden:s-max`   | `< s`      |
| `is-hidden:m`       | `> m`      |
| `is-hidden:m-max`   | `< m`      |
| `is-hidden:l`       | `> l`      |
| `is-hidden:l-max`   | `< l`      |
| `is-hidden:xl`      | `> xl`     |
| `is-hidden:xl-max`  | `< xl`     |
| `is-hidden:2xl`     | `> 2xl`    |
| `is-hidden:2xl-max` | `< 2xl`    |
