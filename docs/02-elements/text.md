The `text` class is a wrapper for common typographic styles used with preformated content.

## Unordered list

```html
<div class="text flow">
  <ul>
    <li>Lorem Ipsum sit dolor amet</li>
    <li>
      Lorem Ipsum sit dolor amet
      <ul>
        <li>Lorem Ipsum sit dolor amet</li>
        <li>Lorem Ipsum sit dolor amet</li>
        <li>Lorem Ipsum sit dolor amet</li>
      </ul>
    </li>
    <li>Lorem Ipsum sit dolor amet</li>
  </ul>
</div>
```
